Detailed Description

template<class... Tags>
class tuples::tagged_tuple< Tags >

A compile time associative container between structs and data.

The template parameters to tagged_tuple are structs with a member type alias, type denoting the type of the data to be associated with the tag. An example tag is:

struct Int {
using type = int;
};

A tagged_tuple is created by, for example

tuples::tagged_tuple<tags::Int, tags::Short0> t0{2, 9};

Elements are retrieved using the get functions as follows:

CHECK(tuples::get<tags::Int>(t0) == 2);
CHECK(tuples::get<tags::Short0>(t0) == 9);
